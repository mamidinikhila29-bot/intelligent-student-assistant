# Intelligent Student Assistant using RAG

## Overview
This project is an AI-powered Student Assistant developed using n8n, Retrieval-Augmented Generation (RAG), Qdrant Vector Database, Google Gemini Embeddings, and Groq Llama.

## Technologies
- n8n
- Qdrant
- Groq Llama 3.1
- Google Gemini Embeddings
- SerpAPI
- JavaScript

## Features
- Document-based Question Answering
- Retrieval-Augmented Generation
- Intelligent AI Routing
- Web Search Integration
- General AI Responses

## Workflow

User Question
      ↓
Qdrant Search
      ↓
Similarity Check
      ↓
Document Found?
   ↓          ↓
 Yes         No
  ↓           ↓
 RAG      AI Router
              ↓
      Search / General
